Binary Search on the Answer

Searching a monotonic predicate rather than a sorted array of values.

patternEasy15 min#binary-search#monotonic-predicate

Intuition

Binary search doesn't need a sorted array; it needs a monotonic predicate over the search space -- false, false, ..., false, true, true, ..., true.

The real requirement

If you can write a check(x) that is False then True exactly once as x increases, you can binary search on x -- even if x was never an array index to begin with.
