Two Pointers on a Sorted Array

Collapsing an O(n²) pair search into O(n) by moving two indices toward each other instead of checking every pair.

patternEasy12 min#two-pointers#arrays#sorted

Intuition

On a sorted array, if the sum at (left, right) is too big, only shrinking right can help; if it's too small, only growing left can. That one-directional argument is what lets both pointers move only forward, ever.

Common Mistakes

Moving both pointers at once

Advancing left and right in the same iteration can skip over the one valid pair entirely -- move exactly one pointer per step.
